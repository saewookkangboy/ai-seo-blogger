## 📊 **시스템 개요**

### 🎯 **프로젝트 목적**
AI를 활용하여 특정 키워드에 대한 해외 자료를 수집, 번역, 요약하고 SEO에 최적화된 블로그 초안을 생성하는 종합적인 콘텐츠 생성 시스템

### 🏗️ **아키텍처**
- **백엔드**: FastAPI (Python 3.13)
- **데이터베이스**: SQLite (SQLAlchemy ORM)
- **프론트엔드**: HTML/CSS/JavaScript (Bootstrap 5)
- **AI 서비스**: OpenAI GPT-4o-mini, Google Gemini 2.0 Flash
- **번역 서비스**: DeepL API (선택사항)
- **배포**: Docker 지원
