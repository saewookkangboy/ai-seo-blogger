# 프로젝트 업데이트 이력

이 문서는 AI SEO Blogger 프로젝트의 주요 변경 이력을 기록합니다.

## 2025년 주요 변경 이력

### 2025.12.XX
- **Google Docs Archive 기능 추가**: 블로그 포스트 생성 시 자동으로 Google Docs로 Archive 저장
  - Google Docs API 통합, Archive 폴더 자동 생성 및 관리
  - Archive 관리 API 엔드포인트 추가 (문서 목록 조회, 수동 Archive, 문서 삭제)
  - HTML 태그 제거 및 깔끔한 텍스트 형식으로 저장
  - 상세한 설정 가이드 및 테스트 스크립트 제공

### 2025.11.26
- **Advanced SEO Guidelines Implementation**: 고급 SEO 가이드라인 구현
  - AEO, AIO, GEO 최적화를 위한 구체적인 프롬프트 추가
  - SEO 가이드라인 업데이트 (2025년 말 트렌드 반영)
  - 답변 길이(40-60단어) 및 인용 권위 메트릭 추가

### 2025.11.26
- **Enhanced Features Implementation**: 향상된 기능 구현
  - Enhanced Mode 엔드포인트 추가 (`/generate-post-enhanced`)
  - AI 분석 통합 (신뢰도 점수, SEO 점수, AI 요약)
  - Gemini 2.0 Flash 명시적 통합
  - 검증 테스트 추가

### 2025.11.25
- **Navigation Improvement**: 네비게이션 개선
  - 관리자 페이지 링크를 버튼 스타일로 변경
  - 시각적 강조 및 사용자 경험 개선

### 2025.11.25
- **Admin Dashboard Optimization**: 관리자 대시보드 최적화
  - 외부 JavaScript 파일 분리 (`admin.js`)
  - 통합 API 호출 함수 및 에러 처리
  - Toast 알림 시스템
  - 코드 품질 및 유지보수성 향상

### 2025.11.25
- **Frontend Performance Optimization**: 프론트엔드 성능 최적화
  - 외부 CSS 파일 분리 (`admin.css`)
  - HTML 파일 크기 15% 감소 (255KB → 217KB)
  - 캐싱 효율성 개선
  - 초기 페이지 로드 시간 20-30% 개선

### 2025.10.22
- **자동 업데이트 시스템 구축**: 전체 서비스의 테스트, 진단, 자동 업데이트, 로그 기록 시스템 구축 완료

### 2025.10.22
- **시스템 진단 완료**: 치명적 오류 1개, 경고 2개 식별

### 2025.01.25
- **Modern SEO Guidelines Implementation**: 현대적 SEO 가이드라인 구현
  - SEO 가이드라인 설정 파일 생성 (`seo_guidelines.py`, 400+ 줄)
  - AI SEO (E-E-A-T 기반 최적화)
  - AEO (ChatGPT, Perplexity 답변 엔진 최적화)
  - GEO (Google SGE, Bing Chat 생성형 엔진 최적화)
  - AIO (Google AI Overviews 최적화)
  - AI Search (통합 AI 플랫폼 최적화)
  - 관리자 대시보드에 SEO 정책 섹션 추가
  - API 엔드포인트 3개 추가

## 2024년 주요 변경 이력

### 2024.06
- **UI/UX 전면 통일**: 네비게이션, 푸터, 폰트, 카드/테이블/버튼 등 모든 페이지 일관화
- **공통 컴포넌트 도입**: 공통 CSS(`common.css`), 공통 네비(`_navbar.html`)
- **관리자 페이지 고도화**: 탭별 대시보드, 키워드/포스트/크롤링/시스템 관리
- **생성 기록 페이지 개선**: 가상 스크롤 제거, 전체 스크롤/전체 리스트 항상 표시
- **포스트 일괄 관리**: 일괄 삭제/내보내기/복원/엑셀 내보내기(선택/전체) 프론트/백엔드/DB 레이어 완비
- **Pydantic v2 마이그레이션**: 직렬화/타입/None 처리 보완
- **대량 데이터 환경 대응**: 성능 최적화 및 버그 수정
- **연관 참고글 추천 정책 고도화**: 입력 정책에 따른 추천 방식 분기
- **AI 기능 확장**: AI 요약, 신뢰도 평가, 연관 키워드/질문/카테고리 추천 옵션화

- **크롤링 현황판**: 문제 사이트 자동 식별
- **실시간 반영 구조**: JS/이벤트/검색/정렬/삭제 등 실시간 반영

- **관리자 대시보드 UI 고도화**: 공통 CSS/폰트/네비게이션/푸터 통일
