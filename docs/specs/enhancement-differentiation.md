# 서비스 고도화 및 차별화 스펙 (역할 기반)

## 1. 유사 서비스 리서치 요약

### 1.1 경쟁 도구 현황 (2024–2025)

| 도구/특징 | 핵심 기능 | 본 서비스 대비 |
|-----------|-----------|----------------|
| **RankYak** | 자동 SEO 블로깅, 키워드 리서치, 1클릭 생성, CMS 연동 | 우리: 타겟 분석 + 스트리밍 생성, AI 윤리·인용 분석 |
| **Jasper** | SEO/AEO/GEO 리라이터, 캔버스·그리드 워크플로우 | 우리: 단일 화면 탭 플로우, Responsible AI 점수 |
| **SEO Writing AI** | 48언어, 벌크 생성, WordPress | 우리: GEO·AIO·AI 윤리 통합, 타겟 분석 연동 |
| **Backsmith** | 키워드 클러스터링, 실시간 SEO 스코어, 경쟁 분석 | 우리: 타겟 분석(키워드/오디언스/경쟁자) + 포스트 생성 연동 |

### 1.2 차별화 포인트 (도입·강화)

- **타겟 분석 → 포스트 작성 연동**: 분석 결과를 “포스트 작성에 적용”으로 한 번에 반영.
- **SEO·GEO·AIO·AI 윤리 일원화**: 생성 결과에 Responsible AI 점수, 출처·인용 분석, GEO/AIO 품질 안내를 한 화면에서 제공.
- **명확한 워크플로우**: 탭 기반(타겟 분석 | 새 포스트 작성) + 통합 결과 영역으로 “분석 → 생성 → 결과” 플로우 단순화.
- **재생성·제어 가능성**: 결과 하단 “다시 생성” 안내로 AI 변동성 제어(UX 베스트 프랙티스 반영).

## 3. 검증 기준

- 타겟 분석 완료 후 “포스트 작성에 적용” 클릭 시 새 포스트 탭으로 전환되고, 주제/문맥 필드에 분석 키워드·컨텍스트가 반영된다.
- 결과 영역에 “다시 생성” 또는 품질 요약 안내가 노출된다.
- 히어로 또는 탭 근처에 SEO·GEO·AIO·AI 윤리 차별화 문구가 노출된다.
