# AI-Powered SEO Blog Post Generator

AI를 활용하여 특정 키워드에 대한 해외 자료를 수집, 번역, 요약하고 SEO에 최적화된 블로그 초안을 생성하는 프로젝트입니다.

## 🚀 주요 기능

### 📊 **핵심 시스템 기능**

| 구분 | 기능명 | 세부 설명 |
|------|--------|-----------|
| **AI 콘텐츠 생성** | OpenAI/Gemini 통합 | OpenAI GPT-4, Gemini 2.0 Flash 기반 블로그 포스트 생성, 스타일별 가이드라인 적용 |
| **AI 콘텐츠 생성** | 향상된 콘텐츠 생성 | AI 추천 키워드, 주요 내용, 핵심 포인트, 실용적 팁, 요약, 신뢰도 평가, SEO 최적화 점수 |
| **AI 콘텐츠 생성** | 콘텐츠 스타일 지원 | 뉴스, 리뷰, 가이드, 요약, 블로그(개인적 인사이트), Enhanced 모드 |
| **AI 콘텐츠 생성** | 분량별 최적화 | 2000자, 3000자, 4000자, 5000자 분량별 AI 프롬프트 최적화 |
| **번역 시스템** | 다중 API 지원 | DeepL, Gemini, OpenAI API 기반 번역, 할당량 과부하 시 자동 fallback |

| **크롤링 시스템** | 사이트별 크롤러 | Search Engine Land, Social Media Today, Marketing Land 등 주요 SEO 사이트 |

| **크롤링 시스템** | Selenium 크롤러 | 동적 콘텐츠 크롤링, JavaScript 렌더링 지원 |
| **크롤링 시스템** | 스마트 크롤러 | AI 기반 콘텐츠 품질 평가, 자동 필터링 |
