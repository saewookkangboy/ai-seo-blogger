# AI-Powered SEO Blog Post Generator

AI를 활용하여 특정 키워드에 대한 해외 자료를 수집, 번역, 요약하고 SEO에 최적화된 블로그 초안을 생성하는 프로젝트입니다.

## 🚀 주요 기능

|------|--------|-----------|
| **AI 콘텐츠 생성** | OpenAI/Gemini 통합 | OpenAI GPT-4, Gemini 2.0 Flash 기반 블로그 포스트 생성, 스타일별 가이드라인 적용 |
| **AI 콘텐츠 생성** | 향상된 콘텐츠 생성 | AI 추천 키워드, 주요 내용, 핵심 포인트, 실용적 팁, 요약, 신뢰도 평가, SEO 최적화 점수 |

```bash
git clone <repository-url>
cd ai-seo-blogger
```

### 2. 가상환경 생성 및 활성화
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

### 3. 의존성 설치
    ```bash
    pip install -r requirements.txt
    ```

### 4. 환경 변수 설정
```bash
cp .env.example .env
```

`.env` 파일을 편집하여 API 키를 설정하세요:
```env
OPENAI_API_KEY=your_openai_api_key_here
DEEPL_API_KEY=your_deepl_api_key_here
GOOGLE_DRIVE_CLIENT_ID=your_google_drive_client_id_here
GOOGLE_DRIVE_CLIENT_SECRET=your_google_drive_client_secret_here
```

### 5. 데이터베이스 초기화
```bash
cd app
python -c "from database import engine; from models import Base; Base.metadata.create_all(bind=engine)"
```

## 🚀 실행 방법

### 개발 서버 실행
```bash
cd app
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 프로덕션 서버 실행
    ```bash
    cd app
uvicorn main:app --host 0.0.0.0 --port 8000
    ```

브라우저에서 `http://localhost:8000`으로 접속하세요.

### Google Drive API 설정 (선택사항)
Google Drive 백업 기능을 사용하려면 추가 설정이 필요합니다:

1. **Google Cloud Console 설정**:
   - Google Cloud Console에서 Google Drive API 활성화
   - OAuth 2.0 클라이언트 ID 및 시크릿 생성
   - OAuth 동의 화면 설정 및 테스트 사용자 등록

2. **자동 설정 스크립트 실행**:
```bash
python3 google_drive_setup.py
```

3. **Google Drive API 테스트**:
```bash
python3 test_google_drive_simple.py
```
